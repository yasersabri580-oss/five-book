## 🖼️ HOW TO USE THE IMAGE PROMPTS

Each gig has 3 gallery image prompts and 1 cover prompt. Use them with:

- **DALL·E 3** (ChatGPT Plus) — paste prompt directly
- **Midjourney** — use `/imagine` command with the prompt
- **Adobe Firefly** — paste in the text-to-image field
- **Stable Diffusion** (AUTOMATIC1111 or ComfyUI) — use as positive prompt

**Recommended sizes:**
- Gallery images: **1280 x 960px** (4:3 ratio)
- Cover image: **1280 x 769px** (16:9 ratio)

## 💡 GENERAL FIVERR TIPS

1. **Response time is king** — respond to every message within 1 hour for your first 30 days. This dramatically boosts your ranking.
2. **Offer a free consultation** — say "Message me before ordering" to build trust and filter serious buyers.
3. **Collect reviews aggressively** — after delivery, politely ask: "If you're happy with the work, a review would mean a lot to me!"
4. **Use Fiverr's Promoted Gigs** — once you have 3+ reviews, invest a small budget in promoted listings.
5. **Stay online** — the Fiverr app's green "online" badge drives more impressions.
6. **Niche down** — four focused gigs outperform one generic "I do everything" gig every time.
