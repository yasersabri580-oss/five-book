# ANIMATION PROMPTS — FORGE INFANTRY DRONE — RUN (ALERT SPRINT)

---

## Instructions for All Run Prompts
- Use the canonical reference image (see `reference_prompt.md`) as the visual source of truth.
- **Only direction and dynamic pose change. Body, arm assignments, and all mechanical details are strictly locked.**
- Run (Alert Sprint): Used for Aggressor behavior — faster movement, more aggressive forward lean (~15°). Larger limb extension than walk.
- Sensor module STILL STABLE — gyroscopic stabilization remains. Body lean does not tilt the sensor module.
- Right arm = cannon arm. Left arm = claw arm. Never swap.
- Foot pads show a more aggressive ground strike — the digit-type lower leg shows more extension.
- Sprite size: 192×192 px per frame. Isometric 2.5D. 30-degree pitch.

Sensor module mount top stable. Do NOT change any visual element or swap arm roles.

Pose: Aggressive forward sprint south. Body lean ~15° forward. Right leg fully extended forward at max stride (larger extension than walk — more ground coverage per step). Right cannon arm swings more aggressively forward than in walk. Left claw arm swings back — fingers slightly extended from air resistance. Sensor module mount STABLE despite body lean. Body lean is aggressive but not unstable — this is a machine, not an organic sprinter.

Background: Transparent PNG.
Art style: Semi-realistic painterly sprite, 2px black outline, dynamic mechanical aggression.
Lighting: Baked isometric top-left directional.
