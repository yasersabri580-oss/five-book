# ANIMATION PROMPTS — KAEL DROS — RUN

---

## Instructions for All Run Prompts
- Use the canonical reference image (see `reference_prompt.md`) as the visual source of truth.
- **Only direction and dynamic pose change. Costume, proportions, and visor design are locked.**
- Run cycle: 6-frame looping cycle. More dynamic than walk — greater limb extension, stronger lean.
- Rifle is held diagonally across body during run — right hand on grip, barrel pointed upward at ~45°, left (mechanical) arm pumps for balance.
- Body leans forward at approximately 15°. Head tilts slightly forward.
- Feet show clear off-ground phase — one foot always airborne in the cycle.
- Sprite size: 192×192 px per frame. Isometric 2.5D. 30-degree pitch.
