# ANIMATION PROMPTS — KAEL DROS — IDLE

---

## Instructions for All Idle Prompts
- Use the canonical reference image (see `reference_prompt.md`) as the visual source of truth.
- **Only the direction changes. Do NOT alter the costume, proportions, silhouette, or face/visor design.**
- All idle animations are looping. First and last frames must match silhouette.
- Subtle breathing movement — chest rises slightly, shoulders drop slightly. Mechanical arm makes a micro-adjustment (finger flex or wrist tilt) once per loop.
- Rifle is held in low-ready (barrel angled down-forward).
- Sprite size: 192×192 px per frame. Isometric 2.5D. 30-degree pitch.
