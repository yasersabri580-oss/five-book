# REFERENCE IMAGE PROMPT — SERGEANT KAEL DROS

---

## Purpose
This prompt generates the **canonical reference image** for Kael Dros.
All animation prompts must reference this image as the source of truth for visual identity.
Do not use this prompt for animations — use `animation_prompts/` for those.
