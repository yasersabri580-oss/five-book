# ANIMATION PROMPTS — RECLAIMER MILITIA SOLDIER — RUN

---

## Instructions for All Run Prompts
- Use the canonical reference image (see `reference_prompt.md`) as the visual source of truth.
- **Only direction and dynamic pose change. Costume, helmet, rubber shoulder armor, bandoliers, and weapon are strictly locked.**
- Run: All-out aggressive charge sprint. Forward lean ~20° (significantly more than SENTINEL's 10°). This character runs TOWARD enemies.
- Rifle held one-handed in right hand, dangling at side or slightly raised — NOT tucked to chest like SENTINEL. The loose rifle is a key visual contrast.
- Left arm (bare, with tattoo) pumps aggressively forward.
- Rubber shoulder armor on left bounces significantly with sprint momentum.
- Mouth: slightly open — this character shouts while charging.
- Exaggerated stride length — legs pump high.
- Sprite size: 192×192 px per frame. Isometric 2.5D. 30-degree pitch.
