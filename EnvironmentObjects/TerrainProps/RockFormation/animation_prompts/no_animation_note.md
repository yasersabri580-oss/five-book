# ANIMATION PROMPTS — ROCK FORMATION

---

## Animation Status

**No animation required.**

Rock formations are fully static terrain objects. They do not react to any gameplay event including explosions, tank collisions, or gunfire.

---

## Static Sprite Variants Required

| Variant | Filename | Description |
|---|---|---|
| Small | `rock_small.png` | Single boulder, 0.75 tile footprint |
| Medium | `rock_medium.png` | 2–3 boulder cluster, 1×1 tile footprint |
| Large | `rock_large.png` | Ridge cluster, 1×2 tile footprint |

All variants are independent sprites placed directly in the scene as SpriteRenderer objects.
No sprite sheet or animation controller needed.
