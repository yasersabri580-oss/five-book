# REFERENCE IMAGE PROMPT — ROCK FORMATION

---

## Purpose
This prompt generates the **canonical reference image** for the RockFormation.
All three size variants must be consistent with this reference in material, color, and style.

---

## Reference Image Generation Prompt

```
2.5D isometric game asset — Rock Formation (three size variants) for the game IRON BREACH.

CAMERA: Isometric 2.5D, 30-degree pitch, south-east facing. Objects shown as standalone sprites on a flat near-black background (#121212) or transparent. No environment, no scene.

ART STYLE: Stylized but gameplay-readable. Crisp 2px black outline. Faceted flat-shading with baked isometric light from top-left. Top faces are lightest. South-facing faces are mid-tone. East-facing and undercut shadow faces are darkest. NOT pixel art. NOT photorealistic. NOT rounded smooth rocks. Angular, faceted stone — a stylized game-asset feel. Consistent with the IRON BREACH isometric environment art style.

REFERENCE SHEET LAYOUT: Three panels side by side on the same image, left to right:

PANEL 1 — SMALL ROCK (0.75 tile footprint):
- A single angular boulder. Roughly 0.75 of an isometric tile in footprint, about 0.6 tiles tall. Jagged top silhouette with 3–4 distinct facets. Flat base. A few small loose rock chips at the base. Subtle lichen patch on the upper-right face — muted grey-green (#3A4030), small, not prominent.

PANEL 2 — MEDIUM ROCK CLUSTER (1×1 tile footprint):
- A cluster of 2–3 angular boulders of slightly varying heights. Footprint fills a full 1×1 isometric tile. The tallest boulder reaches about 1 tile height. Jagged, asymmetric silhouette. The boulders press against each other — not separated. One has a faint crack line across its south face.

PANEL 3 — LARGE ROCK FORMATION (1×2 tile footprint):
- An elongated boulder cluster spanning 1×2 isometric tiles. Lower overall height than the medium cluster but wider. Feels like a natural rock ridge. 4–5 faceted boulders of varying sizes, all pressed together as a continuous mass. Left and right ends taper down toward the ground.

COLOR PALETTE (apply to all three panels consistently):
- Top faces: #5E5850 (warm grey-brown, lightest)
- South faces: #4A4440 (mid-tone grey-brown, base color)
- East/shadow faces: #2A2420 (dark, almost black-brown)
- Crack lines: #1A1410 thin dark lines
- Lichen: #3A4030 small muted grey-green patch, only on one face per rock
- Loose chips at base: same palette as main rock, slightly smaller fragments
- Outline: #0A0A0A (black, 2px)

SCALE REFERENCE: Include a faint isometric tile grid ghost under each panel to show footprint clearly.

OUTPUT: One horizontal reference sheet, three panels, all on same dark background. Each variant labeled: "Small", "Medium", "Large". Clean sprite output ready for use.
```

## Checklist — What This Prompt Establishes

- [x] Camera angle (isometric 30-degree, SE facing)
- [x] Art style (stylized faceted, 2px outline, baked lighting)
- [x] Three size variants (small, medium, large)
- [x] Material (angular stone, faceted, not smooth)
- [x] Color palette (three face tones + outline + lichen)
- [x] Detail level (minimal — chips, one crack, small lichen)
- [x] Scale reference (isometric tile ghost)
- [x] Dark background (clean sprite output)
