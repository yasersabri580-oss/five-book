# ANIMATION PROMPTS — RUINED PILLAR

---

## Animation Status

**No animation required.**

The RuinedPillar is fully static. It represents an already-destroyed architectural element placed permanently in the level. It does not react to gameplay events.

---

## Static Sprite Variants Required

| Variant | Filename | Description |
|---|---|---|
| Tall | `pillar_tall.png` | 1.5 tile height broken column |
| Stub | `pillar_stub.png` | 0.75 tile height broken column |

Optional: `pillar_rubble_base.png` — a separate flat rubble chip sprite placed at base-Y for added ground-level detail (rendered below the pillar in sort order).
